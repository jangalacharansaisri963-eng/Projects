# Trigonometry

Trigonometry studies the relationship between angles and sides of triangles.

---

## Remember

SOH

Sine = Opposite / Hypotenuse

CAH

Cosine = Adjacent / Hypotenuse

TOA

Tangent = Opposite / Adjacent

Also

tan(θ)=sin(θ)/cos(θ)

---

## Special Angles

Angle | sin | cos | tan

0° | 0 | 1 | 0

30° | 1/2 | √3/2 | 1/√3

45° | √2/2 | √2/2 | 1

60° | √3/2 | 1/2 | √3

90° | 1 | 0 | Undefined

# sin(x)

Purpose

Returns the sine of an angle.

Formula

sin(θ)=Opposite/Hypotenuse

Example

sin(30)

=0.5

Used For

• Right triangles

• Physics

• Waves

Remember

SOH

---

# cos(x)

Purpose

Returns the cosine of an angle.

Formula

cos(θ)=Adjacent/Hypotenuse

Example

cos(60)

=0.5

Remember

CAH

---

# tan(x)

Purpose

Returns the tangent of an angle.

Formula

tan(θ)=Opposite/Adjacent

tan(θ)=sin(θ)/cos(θ)

Example

tan(45)

=1

Remember

TOA

---

# asin(x)

Purpose

Returns the angle whose sine is x.

Example

asin(0.5)

=30°

---

# acos(x)

Purpose

Returns the angle whose cosine is x.

Example

acos(0.5)

=60°

---

# atan(x)

Purpose

Example
