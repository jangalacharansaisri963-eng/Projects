# Logarithms

A logarithm answers the question

"To what power must the base be raised to obtain this number?"

---

## Remember

Exponent

Base

Power

Example

10²=100

Therefore

log(100)=2

---

# ln(x)

Purpose

Natural logarithm.

Formula

ln(x)=logₑ(x)

Base

e

Example

ln(e)

=1

Used For

Calculus

Growth

Decay

---

# log(x)

Purpose

Common logarithm.

Formula

log₁₀(x)

Base

10

Examples

log(10)

=1

log(100)

=2

log(1000)

=3

Remember

log = base 10

ln = base e

---

## Common Rules

log(ab)

=log(a)+log(b)

log(a/b)

=log(a)-log(b)

log(aⁿ)

=nlog(a)
