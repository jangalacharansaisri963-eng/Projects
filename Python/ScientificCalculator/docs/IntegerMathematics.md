# Integer Mathematics

---

# gcd(a,b)

Also called

HCF

Greatest Common Divisor

Purpose

Largest number dividing both numbers.

Example

gcd(24,18)

=6

Remember

Largest common factor.

---

# hcf(a,b)

Alias for gcd().

---

# lcm(a,b)

Purpose

Least Common Multiple.

Example

lcm(6,8)

=24

Remember

Smallest common multiple.

---

Relationship

gcd(a,b)×lcm(a,b)=a×b

Useful For

Fractions

Ratios

Algebra
