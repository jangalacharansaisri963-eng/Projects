# Hyperbolic Functions

Hyperbolic functions are similar to trigonometric functions but are based on hyperbolas instead of circles.

These functions are commonly used in

- Engineering
- Physics
- Differential Equations

---

# sinh(x)

Purpose

Hyperbolic sine.

Formula

sinh(x)=(e^x-e^-x)/2

Example

sinh(1)

---

# cosh(x)

Purpose

Hyperbolic cosine.

Formula

cosh(x)=(e^x+e^-x)/2

Example

cosh(1)

---

# tanh(x)

Purpose

Hyperbolic tangent.

Formula

tanh(x)=sinh(x)/cosh(x)

Example

tanh(1)

---

Remember

Hyperbolic functions are NOT angles.
They work with real numbers.
