# Chemistry Calculator

Chemistry Calculator is a Python-based application designed to solve common chemistry calculations.

The purpose of this project is to make chemistry calculations easier by converting important concepts into simple commands. Instead of manually calculating every formula, users can enter values and get accurate results.

The first version focuses on the fundamental topics required for many chemistry problems:

- Mole Concept
- Molarity
- Dilution
- Stoichiometry

These concepts are connected because most chemical calculations use the mole as the central unit.

The mole is a bridge between microscopic particles and measurable quantities. It connects the mass of a substance, the number of particles, gas volume, and chemical reactions.

For example, a given mass can be converted into moles using molar mass. Those moles can then be converted into particles using Avogadro's number or used in stoichiometric calculations.

The calculator uses standard chemistry constants:

Avogadro's number:

\[
6.022 \times 10^{23}
\]

This represents the number of particles present in one mole.

For gases at STP, the molar volume is approximately:

\[
22.4L/mol
\]

The project is designed to be expanded in the future with more chemistry features such as:

- Chemical equation balancing
- Periodic table information
- Gas laws
- Thermochemistry
- Electrochemistry

The goal is to create a complete chemistry calculation system where users can learn concepts while solving problems.
