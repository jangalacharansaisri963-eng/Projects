## Java Scientific Calculator

A Java-based terminal-style calculator project.

Features:

- Custom expression parser
- Arithmetic evaluation
- Bracket handling
- BODMAS/PEMDAS evaluation
- Expandable calculator architecture


The goal is to create a lightweight calculator engine built from scratch using Java.
